# Prodesk IT – Responsive Landing Page

A modern, responsive landing page for **Prodesk IT**, built using **HTML5, CSS3, and Vanilla JavaScript**. This project was developed as part of an internship assignment to demonstrate frontend development skills, responsive design, accessibility, and user experience best practices.

---

## 🚀 Features

### Phase 1 – Base MVP
- Responsive Navigation Bar
- Mobile Hamburger Menu
- Hero Section with Call-to-Action Buttons
- Services Section
- Testimonials Section
- Contact Section
- Footer with Social Links
- Mobile, Tablet, and Desktop Support

### Phase 2 – UI/UX Enhancements
- Dark/Light Theme Toggle
- Theme Preference Persistence using Local Storage
- Sticky Navigation Bar
- Hover Animations and Micro-Interactions
- Smooth Transitions
- Professional UI Design

### Phase 3 – Optimization & Accessibility
- Semantic HTML Structure
- Accessibility Improvements
- Keyboard Navigation Support
- Focus States for Interactive Elements
- Performance Optimizations
- Responsive Layout Improvements

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/prodesk-it-landing-page.git
```

2. Open the project folder:

```bash
cd prodesk-it-landing-page
```

3. Open `index.html` in your browser

OR

Use VS Code Live Server:

```text
Right Click index.html
→ Open with Live Server
```
